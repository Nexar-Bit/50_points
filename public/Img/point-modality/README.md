# Point modality UI assets

Place generated **PNG** images here using the filenames in `docs/POINT_MODALITY_ASSET_PROMPTS.md`.

Components load paths via `src/frontend/lib/config/pointModalityAssets.js`. Strategy picker textures also use `ticket-workflow/strategy-*-btn.png` (see same doc).

Verify files after generation:

```bash
npm run verify:point-modality-assets
```
