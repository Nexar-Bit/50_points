# Hall of Fame promotional assets — AI image prompts

Generate each image separately. Save files into `public/Img/hof/` using the **exact filenames** below. Prefer **PNG with transparency** for icons, overlays, and decorative elements; use **JPG/WebP** for full-card backgrounds.

Recommended settings for most prompts: **4K**, **high detail**, **studio lighting**, **no text unless specified**, **isolated on transparent or pure black background** (as noted per asset).

## After generating images

1. Copy all files to `FRONTEND/public/Img/hof/`.
2. Hard-refresh the browser (or clear cache) and open `/hall-of-fame`.
3. Clear `sessionStorage` key `50points-hof-entry-wizard` to replay the entry carousel.
4. Trigger an achievement card from the profile gallery to preview placement designs.

No code changes are required when filenames match — components load assets automatically via `hofAssets.js`.
